# TimeRadar Store Listing

## Positioning

TimeRadar is an Arabic-first focus companion for Saudi professionals who need to start focused work quickly, protect deep-work blocks, and review daily progress through a simple radar signal.

Primary audience: Saudi professionals, freelancers, consultants, founders, designers, developers, and managers who want a clean focus timer without planning noise.

TimeRadar stays free forever. The optional $1.99 Supporter Pack supports the developer and unlocks a warmer visual theme as a small thank-you; it does not lock timer, task, stats, notification, Arabic, English, or RTL features.
